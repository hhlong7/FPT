# Topic 06: OOP Basics

## Muc tieu
- Hieu class, object, constructor, destructor trong boi canh embedded.
- Uu tien abstraction nho, de test, de du doan chi phi.
- Biet composition thuong tot hon inheritance o firmware.
- Ap dung OOP co kiem soat cho print, IoT va robot.

## Pham vi
- C++ co ban cho embedded.
- Khong chay theo OOP hinh thuc; moi abstraction phai co ly do.
- Lien he voi MISRA C++ va chi phi tai nguyen.

## Ban se hoc duoc gi
- Cach bao dong hanh vi trong class nho.
- Cach quan ly resource bang constructor/destructor.
- Cach xoa copy neu object dai dien hardware/resource.
- Cach dung RAII trong lock, file, peripheral session.

## Case study lien quan
- Printer manager.
- Motor controller.
- Sensor service wrapper.
- File handle va lock guard.
