# 06. Review Checklists

## Type safety
- [ ] Fixed-width types da duoc dung
- [ ] Signed/unsigned da duoc xem xet
- [ ] Intermediate overflow da duoc xu ly

## Memory
- [ ] Buffer lon khong dat tren stack
- [ ] Heap policy ro rang
- [ ] NVM block co version/CRC neu can

## API
- [ ] Input/output/status ro rang
- [ ] `const` correctness hop ly
- [ ] Callback context duoc document

## Concurrency
- [ ] ISR chi lam viec nho gon
- [ ] Queue/flag ownership ro rang
- [ ] `volatile` khong bi lam dung nhu co che dong bo day du

## State/Fault
- [ ] State, event, fault code ro rang
- [ ] Invalid transition co fail-safe path
- [ ] Watchdog va reset reason co evidence

## Testability
- [ ] Module co seam de unit test
- [ ] Time source co the fake
- [ ] Boundary va error path co test case

## Performance
- [ ] Co measurement cho hot path
- [ ] Logging khong pha deadline
- [ ] Queue depth va stack budget co co so
